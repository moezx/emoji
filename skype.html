Skype

Skype has a series of emoji-like emoticons which have their own distinct animated appearance.

A subset of Skype emoticons were available in Microsoft Teams before the 3D versions of Microsoft's new Fluent emoji set began rolling out in February 2022.

The Skype emoticon set differs from the emoji set from parent company Microsoft, which is used as the system-wide emoji font in Windows 10.

Skype uses its own system of shortcodes to enter each emoticon, for instance:
